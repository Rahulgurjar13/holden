# Holden India · Official Website

This is the official enterprise web presence for Holden India (Ghodu Infotech Pvt Ltd), a leading provider of medical infrastructure, oxygen systems, and critical care facilities.

## Tech Stack
- **Framework**: React 18 with Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **UI Components**: Radix UI
